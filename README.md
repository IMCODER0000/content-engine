# 콘텐츠 엔진

인스타·스레드·틱톡에 올릴 **글자 카드**랑 **캐러셀**을 빠르게 뽑으려고 만든 개인 도구입니다.
텍스트만 넣으면 디자인은 알아서 나오고, 한국어에 맞춰서 만들었어요.

## 왜 만들었나

카드 생성기들이 죄다 비슷하게 생겨서(그놈의 캔바 템플릿…) 결과물에서 "찍어낸 티"가 나는 게 싫었습니다.
그래서 **글자를 이미지 위에 제대로 앉히는 것** 하나에 집중해서 직접 만들었어요.
폰트·줄바꿈·여백을 픽셀 단위로 잡아서, 템플릿 느낌 안 나게 하는 게 목표였습니다.

## 만들 수 있는 것

- 인용/명언 카드 — 에디토리얼, 모던, 스위스, 센터
- **기대 vs 현실** 비교짤
- **가짜 속보** (The Onion 스타일)
- **음슴체 그린텍스트** 썰 (`>퇴근함 >치킨 시킴 …`)
- 캐러셀 — 진지 공감 / 고전 명언 풍자 / 이상 vs 현실 / 단계별 짤
- 크기: 인스타 1:1·4:5, 스토리 9:16, X 16:9, 핀터레스트 2:3

전부 **PNG 파일**로 떨어집니다. 그대로 올리면 돼요.

## 써보기

```bash
npm install
npm run fonts      # 폰트 받기 (처음 한 번만)
./run.sh           # 브라우저에서 http://localhost:8080
```

에디터가 뜨면 — 종류 고르고 → 내용 쓰고 → 스타일 누르면 바로 미리보기. 다운로드 누르면 끝.
왼쪽 패널 / 하단 패널은 위에서 토글로 바꿀 수 있습니다.

명령줄로 한 번에 여러 장 뽑고 싶을 때:

```bash
npm run deck -- --bank=satire-gap-kr    # 예시 뱅크로 캐러셀
npm run meme versus                     # 밈 카드
npm run matrix                          # 스타일 조합을 한 장에 모아보기
```

## AI 문구 (선택)

주제만 적으면 문구를 대신 써주는 기능이 있는데, `.env`에 `ANTHROPIC_API_KEY`를 넣어야 켜집니다.
없어도 나머지 기능은 전부 됩니다. 문구는 직접 쓰는 게 대부분이라.

## 어떻게 돌아가나 (궁금하면)

- **렌더링** — 글자는 Satori로 SVG를 그린 뒤 resvg로 PNG, 배경이랑 sharp로 합칩니다. 그림 모델한테 글자를 안 맡기는 게 핵심(글자 안 깨지게).
- **서버** — Node 내장 `http`만. 프레임워크 없음.
- **프론트** — 바닐라 JS 한 파일.
- **폰트** — Pretendard, 나눔명조, Black Han Sans, 주아, 고운돋움 등.

혼자 쓰려고 만든 거라 로그인·계정 같은 건 없습니다. 맥에서 띄워두고 다른 기기에선 Tailscale로 접속해서 씁니다.

## 폴더 대충

```
src/design/     레이아웃·팔레트·서체·렌더링
src/content/    문구 규칙(훅·해시태그·캡션)·풍자/밈 뱅크
src/carousel/   캐러셀 조립
src/server.ts   웹 에디터 서버
public/         에디터 화면
docs/           참고 자료(포맷·리서치 정리)
```

메모 삼아 적어둔 리서치·전략은 `docs/`에 있습니다.
