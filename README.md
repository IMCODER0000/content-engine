# 콘텐츠 엔진

한국어 SNS(인스타그램·스레드·틱톡 등)에 게시할 글자 카드와 캐러셀 이미지를 생성하는 도구다.
텍스트를 입력하면 정해진 타이포그래피 규칙에 따라 이미지가 자동으로 구성된다.

![예시 카드](docs/screenshots/hero.png)

## 개요

명언·인용, 일상 공감형 콘텐츠, 풍자, 밈 등 다양한 형식의 카드와 캐러셀을 하나의 도구에서 제작한다.
생성 이미지의 문자는 이미지 생성 모델이 아니라 자체 타이포그래피 엔진으로 직접 배치하므로, 글자 손상 없이 일관된 품질을 유지한다.
결과물은 모두 PNG로 출력되며, 주요 SNS 규격에 맞춰 크기가 조정된다.

개인 용도로 제작된 도구이며, 로그인·다중 사용자 기능은 포함하지 않는다.

## 지원 형식

- 인용·명언 카드 — 에디토리얼 / 모던 / 스위스 / 센터
- 기대 vs 현실 비교 카드
- 가짜 속보(뉴스 헤드라인) 카드
- 그린텍스트(음슴체) 카드
- 캐러셀 — 진지·공감 / 고전 풍자 / 이상 vs 현실 / 단계별

## 주요 기능

- 레이아웃 11종, 색 팔레트 9종, 서체 6종(임팩트·세리프·라운드·돋움 등)
- 배경 — 단색 / 그라데이션 / 질감 / 이미지 합성(가독성 스크림 포함)
- 출력 규격 — 1:1, 4:5, 9:16, 16:9, 2:3
- 단어 강조(형광펜), 글자 크기 및 색상 조정
- 게시용 캡션 및 해시태그 자동 생성
- 실시간 미리보기 기반 웹 에디터 — 좌측/하단 패널 전환, 스타일 프리셋 및 변형, 프로젝트 저장
- 명령줄 일괄 생성
- (선택) API 키 등록 시 문구 자동 작성

## 기술 구성

- 렌더링 — Satori로 텍스트를 SVG로 생성한 뒤 resvg로 PNG 변환, sharp로 배경과 합성
- 서버 — Node.js 내장 `http` 모듈(외부 웹 프레임워크 미사용)
- 프런트엔드 — 단일 HTML 및 바닐라 JavaScript
- 폰트 — Pretendard, 나눔명조, Black Han Sans, 주아, 고운돋움 등

## 디렉터리 구조

```
src/design/     레이아웃·팔레트·서체·렌더링
src/content/    문구 규칙(훅·해시태그·캡션) 및 풍자/밈 콘텐츠 뱅크
src/carousel/   캐러셀 구성
src/server.ts   웹 에디터 서버
public/         에디터 화면
scripts/        명령줄 도구
docs/           참고 자료 및 스크린샷
```
