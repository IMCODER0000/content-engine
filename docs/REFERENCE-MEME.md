# 밈/풍자 포맷 레퍼런스 (한국 SNS 2025–2026)

> 실제 인기 밈·풍자 포맷 리서치(디시/에펨/X/스레드/인스타 + 고구마팜·캐릿·나무위키). 엔진 코드화용 카탈로그.
> 두 라인: **대놓고(blatant)** 사회·제도 풍자 · **트렌디 밈(subtle)** 포맷 기반 드라이 유머.

## 구현 완료 포맷
- **versus**(기대 vs 현실 / 회사가 말하는 X vs 진짜) — 2단 비교 카드. `npm run meme versus`
- **headline**(가짜 속보 / The Onion) — 빨강 속보 배지 + 진지한 헤드라인 + 데드팬 본문. `npm run meme headline`
- **greentext**(음슴체 `>나임` 썰 + 반전) — 초록 줄 + turn 줄 강조. `npm run meme greentext`
- **gap**(이상 vs 현실, 캐러셀) — tone='gap' 뱅크 `satire-gap-kr`
- **ironic**(가짜 명언 "공자, 아마도", 캐러셀) — tone='ironic' 뱅크 `satire-kr`

## 슬랭/메커니즘 템플릿 (라벨에 끼워 쓰기)
- **거의 ~수준/~급**: "출근하기 싫은 거 거의 군대 재입대 수준"
- **~각**: "비 오네, 칼퇴각" / **~력**: "월요일엔 출근력 0"
- **빌런**: "읽씹 빌런" / **국룰**: "치킨은 다리부터, 국룰임"
- **자조/셀프디스**: "통장 보고 깨달음, 나 미니멀리스트였음"
- **사이다 vs 고구마** · **손절** · **갑분싸** · **~의 정석** · **킹받다** · **갓생(↔갓잠)**
- **럭키비키/원영적 사고**(불운을 행운으로 재해석) · **추구미** · **느좋/내또출**(축약)
- **알빠노/어쩔티비**(무관심 디스) · **~하는 중**(상태 캡션) · **~려고요?**(빈정)

## DO / DON'T (공유됨 vs 노잼)
- 핵심 격언: **"설명이 필요한 드립은 실패한 드립이다"**. 적: 설명충·진지충.
- DO: 자기 타겟(자조) · 데드팬(무표정) · 펀치라인 맨 끝 · 짧게/축약 · 시의성 · 흥한 드립은 쿨하게 버리기.
- DON'T: 농담 설명 · 진지빨기/설교 · 인사이드 조크 · 약자 조롱(=비방) · 자기 펀치 반복 · 낡은 밈 우려먹기 · 억지 텐션.

## 풍자 vs 비방 (계정 안 터지게)
- 한국법: **명예훼손**(특정 사실 적시, 사실이어도 처벌 가능) + **모욕죄**(사실 없이 경멸). 공인의 공적 행위는 관대.
- 안전 전술: ① **행동·정책·제도** 겨냥(특정 개인 X) ② 자기 타겟 + "우리 다" 일반화 ③ **아키타입**("라떼 부장", "○○ 빌런") ④ 의견·패러디 프레임 + 명백한 과장 ⑤ 공익 사안에 앵커 ⑥ 혐오·폭력 금지.
- 플랫폼: 정치 콘텐츠는 삭제보다 **노란딱지(수익제한)**가 기본. 섀도밴은 내용보다 **금지 해시태그·과속 포스팅·자동화** 같은 기계적 신호가 원인.

## 비주얼 스펙
- **치수**: 피드 4:5 **1080×1350**(중앙 1080×1080 세이프존, 상하 ~135px 여백). 단발 펀치는 1:1도 OK.
- **폰트**: Pretendard/Noto Sans(기본 모던) · 어그로체/검은고딕(라우드 자막) · 배민도현/주아(친근) · **굴림체/돋움체(아이러니 레트로 — XP·디시 감성, 일부러 못생긴 게 농담)**. 카드당 1~2개, 최대 3.
- **컬러**: 고대비 흑백(순흑/순백은 피함=싸 보임, 단 날것엔 일부러) · **형광펜 노랑 강조**(한 단어) · 카톡 노랑/회색 · 화면캡처 회색·블루.
- **날것 vs 폴리시드**: 관계성·"진짜 사람" 유머 = 날것(스샷·대화체·못생긴 폰트=신뢰). 권위 후 전복(가짜뉴스·광고패러디) = 폴리시드.

## 캡션/해시태그
- 캡션: **반말**, 짧게(1~3줄), 끝에 "..."(데드팬 여운), 친구 태그("이거 완전 너 @소환"), 자조 사인오프("나만 그런 거 아니지?"). 2026 미세 트렌드: **ㅋㅋㅋ 일부러 빼기**(데드팬이 더 쿨).
- 해시태그: **5~8개**(30개 도배 = 노출 억제). 풀에서 회전: #짤 #밈 #드립 #공감 #웃긴짤 #짤방 #일상 + 풍자엔 #풍자, 주제 태그 1~2.
