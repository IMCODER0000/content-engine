# 전략 보고서: 철학·풍자 이미지 콘텐츠 SaaS(B) + 자체 SNS 성장(A)

> 작성일 2026-06-29 · 상태: 결정 락인 · 데이터: 시장/기술/경쟁 리서치(2025–2026) + 분석
> 플랫폼별 수치·API 한도는 출시 전 공식 문서 재확인 필요 (자주 바뀜).

## 0. 한 줄 결론

**"자동화했지만 자동화처럼 안 보이는" 타이포그래피 콘텐츠 엔진**을 만들고, 그걸로
본인 철학·풍자 계정을 먼저 키워(도그푸딩) 그 청중을 그대로 첫 고객으로 전환하는
**빌드인퍼블릭 플라이휠**이 정답. "AI가 포스트를 만든다"는 이제 기본값이고,
차별점은 **미적 퀄리티 + 니치 깊이 + 사실 검증** 세 가지뿐.

## 1. 시장 신호 4가지

1. **AI 슬롭에 플랫폼이 칼을 빼들었다.** 인스타(2025.12 오리지널 콘텐츠 랭킹 강화),
   유튜브(2025 대량생성 금지), Meta(휴먼 검증). → 싸구려 자동화는 도달이 죽는다.
   역설적으로 "잘 만든 자동화"의 희소가치 상승 = 기회.
2. **기존 도구의 구조적 약점 = 텍스트 렌더링.** 확산모델이 글자를 뭉갠다(특히 200자↑).
   다들 텍스트를 별도 오버레이로 우회 → 템플릿 티. 명언 카드는 텍스트가 전부 → 핵심 빈틈.
3. **가짜 인용(misattribution) 무방비.** AI가 그럴듯한 가짜 명언 생성(2026 실제 사고 다수).
   "검증된 출처 표기" 명언 도구가 사실상 없음 → 철학 니치에 완벽한 차별화.
4. **팔로워 ≠ 돈.** 6만 이하가 30만 달러 벌고 150만이 몇백 달러 버는 사례.
   → 전환·퀄리티 중심. 허영 지표 금지.

## 2. 제품 B

### 포지셔닝
"Anti-slop 타이포그래피 엔진" — 디자이너급 명언/풍자 카드를 브랜드 시스템(정확한 hex·폰트·그리드)
잠가서 배치 생성, 인스타+스레드+틱톡 자동 발행.

### 핵심 기술 결정 (해자)
**텍스트를 확산모델에 맡기지 않는다.** 두 레이어 분리:
- 배경: 이미지 생성 API (텍스트 렌더 강한 순서: Ideogram 4.0 / GPT Image 2 / Nano Banana Pro)
- 텍스트: **Satori(JSX→SVG, <50ms) → resvg → PNG, Sharp 합성** → 픽셀 퍼펙트·브랜드 정확
- 카피: Claude Haiku 4.5(캡션/해시태그) + Sonnet 4.6(헤드라인/명언),
  prompt caching(~90%↓) + Batch API(50%↓)

### 발행 API (출시 성패)
- **스레드**: 가장 친화적·무료. 250포스트/24h, 캐러셀 2~20장. 미디어는 공개 URL 필요(S3/CDN). → 1순위
- **인스타**: 프로+FB페이지+앱심사. JPEG만. ~100포스트/24h. → 2순위
- **틱톡**: 가장 제한적. 미심사 앱은 비공개만. 공개 자동발행은 별도 심사(2~4주). 출시 때 약속 금지
- **지름길**: Ayrshare 한 API로 셋 통합 → MVP 빠르게

## 3. 니치 전략 (철학·풍자, 확장 구조)
- 시작: 스토아 철학(~400% 성장) + 현대생활·테크 풍자. "철학 × 현대 모순/풍자" 각도가 빈틈.
- 확장: 콘텐츠를 `{니치 코퍼스 + 보이스 프롬프트 + 브랜드 키트}` 모듈로 분리 → 주제 추가는 모듈만.
- 한국어 시장: 양질 철학·풍자 희소 → 언어 자체가 해자 가능. 본인 강점이면 1순위 고려.
- 아이덴티티 = 사람의 일. 매 카드의 아이디어·각도는 본인, 제작은 AI. (슬롭 페널티 회피의 핵심)

## 4. 수익화 (순서)
| 단계 | 시점 | 수익원 |
|---|---|---|
| 1. 청중 자산화 | 0~3개월 | 팔로워·이메일 리스트 (빌드인퍼블릭 서사) |
| 2. 디지털 상품 | 3~6개월 | 월페이퍼팩·노션템플릿·전자책·프린트(Gumroad/Stan) |
| 3. SaaS(B) 판매 | 6개월~ | Solo $19~29 / Operator·Agency $49~99, 투명 정액제 |

플라이휠: 콘텐츠가 도구 홍보 → 도구가 콘텐츠 양산 → 청중이 고객.

## 5. 리스크 & 대응
1. AI 슬롭 페널티 → 사람 큐레이션 큐 + 독창적 타이포
2. 가짜 인용 → 검증 코퍼스 + 출처 표기를 셀링포인트로
3. 틱톡 자동발행 심사/규제 불확실 → 수동/반자동, 자동화 약속 보류
4. 저가 SaaS WTP·포화 → 차별점이 실제로 보여야 함
5. 저작권 → 퍼블릭도메인 철학 원전 위주로 시작

## 6. 30/60/90일
- **1–30일**: 아이덴티티 확정, 각도 30개 수기, 도구 없이 반자동 20장 → 매일 발행, 무료 월페이퍼팩으로 리스트 시작
- **31–60일**: MVP 코드화(배치+텍스트레이어+승인큐+스레드 자동발행), 빌드인퍼블릭 시작
- **61–90일**: 첫 디지털 상품 판매, 인스타 캐러셀, 니치 모듈 2번째 추가, SaaS 랜딩/대기자

## 부록: 경쟁 도구 요약
