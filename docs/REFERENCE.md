# 인기 콘텐츠 레퍼런스 (2025–2026)

> 실제 인기 철학·스토아·풍자 계정 분석에서 추출한 패턴. 엔진에 코드로 반영됨.
> 참고 계정: @dailystoic, @orangebook_, @stoicism, @thestoicvibe, @philosophyterm(Threads),
> KR 커뮤니티 #명언스타그램 / #인생명언스타그램 / #명언글귀모음, @gpmkt.kr.
> 주의: 벤치마크 수치는 일반 캐러셀 연구(벤더 블로그) 기반 — 패턴은 높은 신뢰, 정확한 %는 방향성으로.

## 1. 훅 (표지 첫 줄) → `src/content/hooks.ts`
첫 슬라이드가 성과의 ~80%. 0.25s, ≤10단어, 1줄. **교훈이 아니라 긴장.**
"첫 1초는 정보가 아니라 긴장감"(@gpmkt.kr). 약한 훅은 인게이지먼트 ~73% 손실.
12개 earnest 템플릿(H1~H12) + 풍자 2개(S1 가짜출처, S2 반전). 계정당 3~4개 로테이션.

## 2. 캐러셀 시퀀스 → `src/carousel/deck.ts`
캐러셀은 현재 최고 인게이지먼트 포맷. 스위트스팟 7~10장, 슬라이드당 ≤20단어·1아이디어.
검증 순서(엔진 반영):
```
표지(훅) → 긴장 → 전환 → 원문(히어로, 4~5번) → 적용 → 재훅(스크린샷 슬라이드) → CTA
```
- **원문은 표지가 아니라 중간**에. 표지는 "스와이프"를 팔고, 원문 노출은 스와이프 깊이를 죽임.
- 출처는 작게, 원문 아래. **재훅 슬라이드**(가장 인용하고픈 한 줄)는 DM/스크린샷 유발 = 공유 엔진.
- 풍자 톤: 4~6장, 진지한 형식 그대로 흉내 + 마지막에 반전(bathos). CTA 없음/반전 CTA.

## 3. 캡션 → `src/content/caption.ts`
스켈레톤: 훅 줄(표지와 다른 표현) → 본문 40~120단어·한 생각 → 답글 유도 → 해시태그.
- 한 생각 = 답글 ~2.3배, 명시적 답글 유도 = 댓글 ~2.8배. "어느 줄이 와닿았나요?" 등.
- 구체적 디테일 > 막연한 감정. KR: 원문을 캡션에 반복 + CTA 압박 낮춤 + 세로 줄바꿈.

## 4. 해시태그 → `src/content/hashtags.ts`
EN: 8~15개 = 광범위 2 + 중간 4 + 니치 6. KR: "~스타그램" 앵커 2개 필수(#명언스타그램, #인생명언스타그램).

## 5. 비주얼 (이미 시스템 반영)
폰트 2개 이하, 한 줄이 히어로, 여백=프리미엄. 다크+세리프(프리미엄 스토아) / 크림+세리프(감성).
세리프=권위/원문, 그로테스크/산세=훅. 장식 폰트 금지(=캔바 슬롭). 그리드 일관성=브랜드 신호.
포트레이트 1080×1350 권장(추후 반영 예정 — 현재 1:1).

## 6. 타이밍
캐러셀: 화·수·목 오전 8~11시(9시 스와이프 완료율 2~3배). 릴스는 저녁 7~9시. 금·토 회피.
첫 30분~2시간이 모멘텀 창. 일관성 > 양.

## 7. 슬롭(도달 죽임) → `src/content/slop.ts`
목 가다듬기 도입부("Here's a thought…", "In today's fast-paced…") 자동 거부.
막연한 감정, 한 슬라이드 다중 아이디어, 20단어 초과, 폰트 3개+, 해시태그 덤핑(30개), 표지에 교훈 노출 — 전부 페널티.
규칙: 모든 포스트에 사람 고유의 디테일/관점 1개.
